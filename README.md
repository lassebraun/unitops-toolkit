# unitops-toolkit

Toolkit for chemical engineering unit operations.

## Simulations

### BET Adsorption Simulation
A 2D Monte Carlo simulation of molecular adsorption.
- **Chemisorption**: The first layer adsorbs with high probability.
- **Physisorption**: Subsequent layers build upon the first with lower probability, simulating the multilayer formation characteristic of the BET model.

Run it with:
```bash
python simulations/bet_adsorption.py
```
